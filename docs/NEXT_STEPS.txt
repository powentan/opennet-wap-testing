üîß TROUBLESHOOTING
================================================================================

Appium won't start?
  lsof -i :4723                    # Check if port is in use
  kill -9 <PID>                    # Kill process
  ./run-emulator-tests.sh start    # Restart

iOS Simulator won't boot?
  xcrun simctl list devices available           # List simulators
  xcrun simctl boot "iPhone SE (3rd generation)" # Boot manually

Android Emulator won't start?
  ~/Library/Android/sdk/emulator/emulator -list-avds    # List AVDs
  ~/Library/Android/sdk/emulator/emulator -avd Medium_Phone_API_35  # Start

Tests failing?
  curl http://localhost:4723/status  # Check Appium is running
  appium driver list                 # Check drivers installed
